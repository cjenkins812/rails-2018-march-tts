* I want to create a new branch called myNew.  How would I do that?
    -You can either do one of the following
      -input ' git branch <branch name>'
      -or git checkout -b <branch name>''

* How do you change branches (Example: change from master branch to myNew branch)?
    - You can view branches by inputting ' git branch', the object preceeded by a '*' is the currently viewed branch
    - When you want to change branches, you can input ' git checkout <branch name>'
